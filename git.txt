git clone: sirve para realizar una copia remota del repositorio. Inidcando la dirección del repositorio actual y del repositorio destino. 
git status: indica la rama en la que estás: main. La rama está aactualizada. Y que no hay cambios pendientes de ser actualizados
git add .: indica que se añaden todos los archivos.
git commit: Crea un comit con los archivos y le asigna un nombre. He elegido homepage ya que es la página principal. 
git push: Se sube a tu repositorio de github y se guarda en él (en tu repositorio remoto)
git checkout: Sirve para ver si se ha actualizado tu repositorio.
